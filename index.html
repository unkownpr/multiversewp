MultiverseWP macOS 14+

Native macOS WhatsApp client with multiple accounts and an embedded MCP server for Claude and other AI assistants.

Install

  1. Download the latest DMG from the Releases page.
  2. Drag MultiverseWP.app to /Applications.
  3. Right-click → Open the first time (Gatekeeper bypass for unsigned builds).

Auto-update

The app checks appcast.xml in this folder.
